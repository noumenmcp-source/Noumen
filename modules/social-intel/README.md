# modules/social-intel/ — соц-разведка / SMM-интент (US)

**Оффер:** понять, что целевая аудитория хочет купить, чем живёт, о чём думает.

**Скоуп:** сбор соцсигналов (YouTube/TikTok/X/Reddit), анализ комментариев, тренды, ICP-research,
интент-сигналы — **tenant-scoped** (под аудиторию конкретного клиента, не владельца).

**Compliant-сбор:** провайдерские API / Apify-акторы, без логин-скрейпа и кастомного обхода
антибота. Без секретов в отчётах. Все находки — с source URL.

**Переиспользование:** коллекторы из RF-проекта YOUTUBE (нативно US-ориентирован) →
переупаковать из owner-tool в per-client SaaS-модуль.
