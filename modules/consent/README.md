# modules/consent/ — согласие и право (US)

**Оффер:** управление куками и юр-тонкостями: баннер согласия, preference-центр, документы.

**Скоуп:** US-CMP — consent-баннер, preference-центр, «Do Not Sell or Share», обработка
GPC-сигнала, неизменяемый журнал-доказательство, генерация документов (Privacy Policy, ToS).

**US-право:** CCPA/CPRA, штатные законы (VA/CO/CT/UT…), CAN-SPAM, TCPA. ⚠️ Не юр-консультация —
ревью US-юристом. Карта: [../../docs/COMPLIANCE-US.md](../../docs/COMPLIANCE-US.md).

**Несущая зависимость, не поздний апселл.** В US сбор чувствительных данных ядром без живого
consent-слоя = юр-экспозиция → модуль обязателен в фазе 1.

**Переиспользование:** закон-агностичная enforcement-логика + ledger-механика (hash-chain+
Ed25519) переносятся из RF-проекта soglasie. Весь 152-ФЗ-контент/документы — **выкинуть**,
пересобрать под US-право.
