# modules/automation/ — автоматизация соцсетей + мессенджеров (US)

**Оффер:** автоматизировать всю активность в соцсетях и мессенджерах.

**Скоуп:** авто-постинг/планирование в соцсетях, авто-ответы и сценарии в мессенджерах
(чаты/боты), оркестрация через движок автоматизации (Temporal/n8n) — мультитенант.

**US-право:** мессенджер-маркетинг под **TCPA** (prior express consent на маркетинговые
сообщения). Без согласия маркетинг-сообщения нельзя — завязка на модуль consent.

**Переиспользование:** наименее готовый модуль (~10–30%). Паттерн чат-ботов есть, но
productized мультитенант-каналы (WhatsApp/Telegram/IG/email) строятся в основном с нуля.
