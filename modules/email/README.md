# modules/email/ — ИИ-email-маркетинг (US)

**Оффер:** письмо под каждого получателя, а не шаблон на всех. ИИ собирает текст под профиль.

**Скоуп:** автосценарии (welcome, брошенная корзина, реактивация, 20+ триггеров), per-profile
ИИ-персонализация, гейт качества письма до отправки, доставляемость.

**US-право:** CAN-SPAM (unsubscribe ≤10 дней, физ-адрес отправителя, честные subject/from),
double-opt-in. US-ESP (Resend/SES US). Контент — English.

**Переиспользование:** движок класса Dittofeed + шаблонизатор + триггеры переносятся из
RF-движка, очищенные от RU-ESP и русского контента. Самый готовый модуль → revenue-клин.
