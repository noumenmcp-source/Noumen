# platform/ — аккаунт и мультитенант (US)

Несущий фундамент экосистемы. Превращает набор движков в SaaS.

**Скоуп:** signup/login, мультитенант (OIDC), изоляция тенантов, RBAC, биллинг (usage/seat),
единая веб-консоль «все данные», коннекторы/SDK для установки на сайт клиента.

**Статус:** фаза 1 (блокер №1). В исходных RF-движках мультитенанта нет — строится здесь.

**US-only.** Хостинг US-облако. Никаких RF-доступов/серверов.
