# core-cdp/ — ядро данных (US)

Хребет: first-party сбор → identity-graph → единый профиль → сегменты/скоринг → консоль.

**B2B-профиль:** фирмографика (компания, размер, индустрия) · интент (сигналы покупки) ·
поведение · соцсигналы. IP→компания (путь Clearbit/6sense/RB2B), не потребительская демография.

**Сбор только с consent-gating** (CCPA/CPRA) — см. [../docs/COMPLIANCE-US.md](../docs/COMPLIANCE-US.md).

**Переиспользование:** паттерн ingest-gateway + identity + профиль переносится из RF-движка,
очищенный от РФ-специфики (Beget/ES-РФ/RU-конфиги). US-облако, полноценный стек без RF-трилеммы.
