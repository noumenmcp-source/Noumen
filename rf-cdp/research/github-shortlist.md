## CDP / customer data infrastructure

- [PostHog/posthog](https://github.com/PostHog/posthog) - all-in-one customer platform benchmark.
- [rudderlabs/rudder-server](https://github.com/rudderlabs/rudder-server) - core event routing / CDP backbone.
- [Tracardi/tracardi](https://github.com/Tracardi/tracardi) - API-first composable CDP engine.
- [airbytehq/airbyte](https://github.com/airbytehq/airbyte) - warehouse sync / reverse-ETL backbone.
- [snowplow/snowplow](https://github.com/snowplow/snowplow) - customer data infrastructure reference.

## Email marketing / newsletter / delivery tooling

- [knadh/listmonk](https://github.com/knadh/listmonk) - lean newsletter and mailing list manager.
- [mautic/mautic](https://github.com/mautic/mautic) - broad marketing automation reference.
- [postalserver/postal](https://github.com/postalserver/postal) - mail delivery platform.
- [mjmlio/mjml](https://github.com/mjmlio/mjml) - responsive email template framework.
- [useplunk/plunk](https://github.com/useplunk/plunk) - modern open-source email platform.
