## Proven end-to-end (the whole loop, each link live)
site event -> ingest-gateway -> Dittofeed profile (ClickHouse) -> ml-content-worker (Flot gen +
Validator) -> traits back -> segment (computed) -> broadcast -> per-recipient render (zavod template,
real catalog images) -> Resend -> inbox; unsubscribe honored natively.

## Honest gaps / debt
- **Storefront tracker not installed** in aiml-mag (dev's task; flag-gated, deploy from pm99lvl).
  Endpoint already cut over to the permanent gateway (S2); waiting on flag flip + acceptance #2/#3.
- **catalog_personalize NOT wired into the loop** — email showcase is static 3 sections, not per-profile.
- **Email images are webp** — Gmail ok, Apple Mail/Outlook may not render; prod should serve png/jpg.
- **Multi-tenant (OIDC) not set up** — single-tenant Default workspace only. Needed for true SaaS resale.
- **Hosting = laptop only** — Vultr GostWheel (3.8GB/~1.6GB free, runs ES+Odoo) can't fit ~4GB CDP stack;
  server deferred. Local+cloudflared-tunnel for the test.
- **No bounce/complaint feed, no double-opt-in** — deliverability hardening pending (needs prod ESP).
- **No CI; gateway not dockerized; worker config.py unreviewed; product-level (not section) personalization absent.**
- **Stage-2 CDP spine (Unomi vs custom) not started** (by design).
